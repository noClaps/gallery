# Aperturic Focus

A website to showcase some cool photos I've taken.

## Build instructions

1.  Clone the repository.

    ```sh
    git clone https://github.com/noClaps/gallery.git
    cd gallery
    ```

2.  Install dependencies.

    ```sh
    bun install
    ```

3.  Build the site.

    ```sh
    bun run build
    ```

    This will build the site and place it in `dist/`.
