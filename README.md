# Aperturic Focus

A website to showcase some cool photos I've taken.

## Build instructions

1.  ```sh
    git clone https://github.com/noClaps/gallery.git
    cd gallery
    ```

2.  ```sh
    bun install
    bun run build
    ```
    This will build the site and place it in `dist/`.

3.  ```sh
    bun serve
    ```
    This will start a development server from the built files.
